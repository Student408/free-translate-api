Free Translate API Documentation

A simple and effective service for translating text between multiple languages.

API Endpoints

POST /translate

Translates the provided text to the specified language.

Request Body:

{
    "text": "Hello, world!",
    "to": "es"
}

Response:

{
    "translatedText": "¡Hola, mundo!",
    "status": true,
    "message": ""
}
GET /health

Check the health status of the API.

Response:

{
    "status": "healthy"
}

    • Testing the API

    Using cURL

    Test the translation endpoint with cURL:

    curl -X POST -H "Content-Type: application/json" -d "{\"text\": \"Hello\", \"to\": \"ja\"}" http://localhost:8000/translate

    Expected Response:

    {
    "translatedText": "Halo",
    "status": true,
    "message": ""
}

    Check the health endpoint:

    curl http://localhost:8000/health

    Streamlit Interface

    Setup:

    1. Ensure you have Python installed
    2. Install the required packages:
    pip install streamlit requests

    Running the Interface:

    python app.py

    Access the interface at http://localhost:8501
